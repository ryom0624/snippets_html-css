position element

CSSのposition:「absolute」と「relative」とは

Ref : UX MILK

box1(relative)
inner(absolute)
box2(absolute/t:250,l:640)
box3(relative)

position要素
relative:自分の本来の場所から指定して移動
absolute:windowの左上から絶対指定で移動
※親要素がrelativeかかっていると、子要素にabsoluteしても親要素からの移動になる。

Code

            
              <ul class="sample">
                  <li>リスト1</li>
                  <li>リスト2</li>
                  <li>リスト3</li>
                  <li>リスト4</li>
              </ul>

Difinition List

DL : HTMLクイック・リファレンス

Ref : HTML クイックリファレス

<dl></dl>タグ
Difinition Listの略
<dt></dt>タグ
定義する用語
<dt></dt>タグ
その説明

dl/dt/ddタグ
<dl>はDefinition Listの略<dl></dl>の範囲が定義リストであることを表します。
<dt></dt>に定義用語、
<dd></dd>に説明を記述。
<dl></dl>の中に複数並べることができる。

Code

              
                <ul class="sample">
                    <li>リスト1</li>
                    <li>リスト2</li>
                    <li>リスト3</li>
                    <li>リスト4</li>
                </ul>

float

CSSの【float】についてちょっと本気出して説明してみた。

Ref : たねっぱ

[パターン1]box3にclearを指定していない場合

box1
box2
box3

[パターン2]box3にclearを指定していない場合

box1
box2
box3

float
floatを指定する際、指定されたboxは宙に浮いているような状態になるので、
box3がbox1の中に入り込んでしまっている。box3の方が少し長いのではみ出している。
一方box3にclear:bothを指定しているパターン2の方は、box2のfloatも解除しているので、
box2より下に配置ができる。

Code

            
              <ul class="sample">
                  <li>リスト1</li>
                  <li>リスト2</li>
                  <li>リスト3</li>
                  <li>リスト4</li>
              </ul>
