Ref : UX MILK

box1(relative)

position要素
relative:自分の本来の場所から指定して移動
absolute:windowの左上から絶対指定で移動
※親要素がrelativeかかっていると、子要素にabsoluteしても親要素からの移動になる。

Code

        
          <ul class="sample">
              <li>リスト1</li>
              <li>リスト2</li>
              <li>リスト3</li>
              <li>リスト4</li>
          </ul>

Difinition List

DL : HTMLクイック・リファレンス

Ref : HTML クイックリファレス

<dl></dl>タグ
Difinition Listの略
<dt></dt>タグ
定義する用語
<dt></dt>タグ
その説明

dl/dt/ddタグ
<dl>はDefinition Listの略<dl></dl>の範囲が定義リストであることを表します。
<dt></dt>に定義用語、
<dd></dd>に説明を記述。
<dl></dl>の中に複数並べることができる。

Code

          
            <ul class="sample">
                <li>リスト1</li>
                <li>リスト2</li>
                <li>リスト3</li>
                <li>リスト4</li>
            </ul>
